## üöÄ Setup Instructions

### Prerequisites (Install Once)
1. **Node.js** (version 18 or higher)
2. **Expo CLI**: `npm install -g @expo/cli`
3. **Expo Go app** on your mobile device (iOS/Android)

### Project Setup
```bash
# 1. Navigate to project directory
cd "Project 01 Sample Answer"

# 2. Install dependencies
npm install

# 3. Start the development server
npx expo start
```

## üì± Running the App

### Option 1: Expo Go (Recommended for Demo)
1. **Start server**: `npx expo start`
2. **Scan QR code** with Expo Go app
3. **App loads directly** on your device

### Option 2: Web Browser
1. **Start server**: `npx expo start`
2. **Press 'w'** in terminal
3. **App opens** in browser

## üéì Demo Flow for Students

### 1. Home Screen (2 minutes)
- Show the clean, professional interface
- Point out the statistics (Photos Taken, Projects Created)
- Demonstrate the "Recent Projects" section
- Click "View All" to show albums

### 2. Camera Functionality (3 minutes)
- Navigate to Camera tab
- Show the camera interface
- Take a photo (if device has camera)
- Demonstrate the responsive back/home buttons

### 3. Gallery & Image Selection (2 minutes)
- Go to Gallery tab
- Show how to select images
- Demonstrate the "Add" button functionality
- Show the albums view

### 4. Photo Editor - The Star Feature (5 minutes)
- Select an image to edit
- **Show Undo/Redo system** (impressive feature!)
- Demonstrate each tool:
  - **Adjustments**: Brightness, Contrast, Saturation
  - **Filters**: Grayscale, Sepia, Vintage, Sharp
  - **Drawing Tools**: Brush sizes, colors, tools
  - **Text Tools**: Fonts, colors, sizes
- Show the history tracking system

### 5. Community Features (2 minutes)
- Navigate to Community tab
- Show how to create posts
- Demonstrate the like functionality

## üîß Troubleshooting

### Common Issues
1. **"Metro bundler" errors**: Restart with `npx expo start --clear`
2. **Slow loading**: Check internet connection
3. **App crashes**: Restart Expo Go app

### If Something Goes Wrong
- **Restart server**: `Ctrl+C` then `npx expo start`
- **Clear cache**: `npx expo start --clear`
- **Check dependencies**: `npm install`
